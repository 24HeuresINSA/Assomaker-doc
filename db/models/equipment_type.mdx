Un type d'équipement permet aux utilisateurs de regrouper et facilement trouver n'importe quel
équipement.

### Champs

#### Signalétique

+ Si le champs est requis, il est signalé par <Required />.
+ Si il a une valeur par défaut, il est signalé par <DefaultValue>valeur par défaut</DefaultValue>

#### Attributs

Nom | Type
--- | ---
<Name>name</Name>  <Required /> | string

### Exemple de réponse

```json
{
  "id": 1,
  "name": "Véhicule",
  "createdAt": "2021-03-03T21:59:21.000Z",
  "updatedAt": "2021-03-03T21:59:21.000Z"
}
```
