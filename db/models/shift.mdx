Un créneau est une plage horaire, à laquelle est associée une valeur de *'charisme'*, permettant
de noter la difficulté de celui-ci.

### Champs

#### Signalétique

+ Si le champs est requis, il est signalé par <Required />
+ Si il a une valeur par défaut, il est signalé par <DefaultValue>valeur par défaut</DefaultValue>
+ Si un champs est une clé étrangère, elle est signalée par <ForeignKey to="shift" label="Créneau" /> 

#### Attributs

Nom | Type
--- | ---
<Name>start_date</Name>  <Required /> | date 
<Name>end_date</Name>  <Required /> | date 
<Name>charisma</Name>  <Required /> | integer
<Name>shift_category_id</Name> <Required /> <ForeignKey to="shift_category" label="Catégorie de créneau" /> | integer

{

  "start_date": "2019-05-19T10:00:00.000Z",

  "createdAt": "2021-03-03T21:59:21.000Z",
