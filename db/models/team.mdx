Une équipe regroupe plusieurs utilisateurs. Un utilisateur peut être dans plusieurs équipes.

### Champs

#### Signalétique

+ Si le champs est requis, il est signalé par <Required />
+ Si il a une valeur par défaut, il est signalé par <DefaultValue>valeur par défaut</DefaultValue>
+ Si un champs est une clé étrangère, elle est signalée par <ForeignKey to="shift" label="Créneau" /> 

#### Attributs

Nom | Type
--- | ---
<Name>name</Name>  <Required /> | string 
<Name>confidenceLevel</Name>  <Required /> | integer <DefaultValue>0</DefaultValue>
<Name>priority</Name>  <Required /> | integer <DefaultValue>10</DefaultValue>
<Name>color</Name>  <Required /> | string <DefaultValue>#888888</DefaultValue>
<Name>supervisorId</Name> <Required /> <ForeignKey to="user" label="Responsable" /> | integer

### Exemple de réponse

```json
{
  "id": 1,
  "name": "INSA",
  "confidenceLevel": 0,
  "priority": 10,
  "color": "#888888",
  "createdAt": "2021-03-03T21:59:21.000Z",
  "updatedAt": "2021-03-03T21:59:21.000Z",
  "supervisorId": 1
}
```
