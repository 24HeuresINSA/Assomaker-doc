Un utilisateur peut aller du bénévole qui n'as accès qu'as son emploi du temps, jusqu'au super admin
qui peut modifier l'entièreté d'Overbookd.
### Champs

#### Signalétique

+ Si le champs est requis, il est signalé par <Required />
+ Si il a une valeur par défaut, il est signalé par <DefaultValue>valeur par défaut</DefaultValue>
+ Si un champs est une clé étrangère, elle est signalée par <ForeignKey to="shift" label="Créneau" /> 

#### Attributs

Nom | Type
--- | ---
<Name>firstName</Name>  <Required /> | string 
<Name>lastName</Name>  <Required /> | string 
<Name>surname</Name>   | string 
<Name>birthday</Name>   | dateonly 
<Name>phoneNumber</Name>   | string(15) 
<Name>email</Name>  <Required /> | string 
<Name>licenceDate</Name>   | date 
<Name>licenceScanUrl</Name>   | string 
<Name>profilePicUrl</Name>   | string <DefaultValue>images/defaultPicUrl.jpg</DefaultValue>
<Name>tshirtSize</Name>   | enum("xs", "s", "m", "l", "xl") 
<Name>alcoholicBeverageConsumption</Name>  <Required /> | integer <DefaultValue>0</DefaultValue>
<Name>foodAndBeverageConsumption</Name>  <Required /> | integer <DefaultValue>0</DefaultValue>
<Name>balance</Name>  <Required /> | decimal(6, 2) <DefaultValue>0</DefaultValue>
<Name>comment</Name>   | text("medium") 
<Name>experience</Name>   | text("medium") 
<Name>incapacity</Name>   | text("medium") 
<Name>validityStatus</Name>  <Required /> | boolean <DefaultValue>false</DefaultValue>
<Name>keycloakUserId</Name>  <Required /> | string <DefaultValue>null</DefaultValue>
<Name>specialtyId</Name> <Required /> <ForeignKey to="specialty" label="Spécialité" /> | integer
