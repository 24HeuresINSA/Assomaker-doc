Une activité est une partie de l'évènement comportant plusieurs tâches.

### Champs

#### Signalétique

+ Si le champs est requis, il est signalé par <Required />
+ Si il a une valeur par défaut, il est signalé par <DefaultValue>valeur par défaut</DefaultValue>
+ Si un champs est une clé étrangère, elle est signalée par <ForeignKey to="shift" label="Créneau" /> 

#### Attributs

Nom | Type
--- | ---
<Name>name</Name>  <Required /> | string 
<Name>description</Name>   | text("medium") 
<Name>startDate</Name>  <Required /> | date 
<Name>endDate</Name>  <Required /> | date 
<Name>contractorName</Name>   | string 
<Name>contractorPhone</Name>   | string 
<Name>contractorMail</Name>   | string 
<Name>contractorComment</Name>   | string 
<Name>contractorPresentOnEvent</Name>   | string
<Name>supervisorId</Name> <Required /> <ForeignKey to="user" label="Responsable" /> | integer
<Name>eventId</Name> <Required /> <ForeignKey to="event" label="Evènement" /> | integer
