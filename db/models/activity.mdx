Une activité est une partie de l'évènement comportant plusieurs tâches.

### Champs

#### Signalétique

+ Si le champs est requis, il est signalé par <Required />
+ Si il a une valeur par défaut, il est signalé par <DefaultValue>valeur par défaut</DefaultValue>
+ Si un champs est une clé étrangère, elle est signalée par <ForeignKey to="shift" label="Créneau" /> 

#### Attributs

Nom | Type
--- | ---
<Name>name</Name>  <Required /> | string 
<Name>description</Name>   | text("medium") 
<Name>start_date</Name>  <Required /> | date 
<Name>end_date</Name>  <Required /> | date 
<Name>contractor_name</Name>   | string 
<Name>contractor_phone</Name>   | string 
<Name>contractor_mail</Name>   | string 
<Name>contractor_comment</Name>   | string 
<Name>contractor_present_on_event</Name>   | string
<Name>supervisor_id</Name> <Required /> <ForeignKey to="user" label="Responsable" /> | integer
<Name>event_id</Name> <Required /> <ForeignKey to="event" label="Evènement" /> | integer
