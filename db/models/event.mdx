Un évènement d'Overbookd regroupe un ensemble d'activité, de tâches et de bénévoles.

### Champs

#### Signalétique

+ Si le champs est requis, il est signalé par <Required />.
+ Si il a une valeur par défaut, il est signalé par <DefaultValue>valeur par défaut</DefaultValue>

#### Attributs

Nom | Type
--- | ---
<Name>name</Name>  <Required /> | string 
<Name>description</Name>   | text("medium") 
<Name>start_date</Name>  <Required /> | date 
<Name>end_date</Name>  <Required /> | date 
<Name>logo_url</Name>   | string <DefaultValue>event_logo_default.png</DefaultValue>

### Exemple de réponse

```json
{
  "id": 1,
  "name": "24h de l\"INSA",
  "description": "le plus grand festival étudiant de france",
  "start_date": "2019-05-17T18:00:00.000Z",
  "end_date": "2019-05-19T21:30:00.000Z",
  "logo_url": "event_logo_default.png",
  "createdAt": "2021-03-03T21:59:21.000Z",
  "updatedAt": "2021-03-03T21:59:21.000Z"
}
```
