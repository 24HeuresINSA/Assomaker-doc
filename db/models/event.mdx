Un évènement d'Overbookd regroupe un ensemble d'activité, de tâches et de bénévoles.

### Champs

#### Signalétique

- Si le champs est requis, il est signalé par <Required />.
- Si il a une valeur par défaut, il est signalé par <DefaultValue>valeur par défaut</DefaultValue>

#### Attributs

| Nom                                 | Type                                                       |
| ----------------------------------- | ---------------------------------------------------------- |
| <Name>id</Name> <Required />        | number                                                     |
| <Name>name</Name> <Required />      | string                                                     |
| <Name>description</Name>            | text("medium")                                             |
| <Name>startDate</Name> <Required /> | date                                                       |
| <Name>endDate</Name> <Required />   | date                                                       |
| <Name>pictureUrl</Name>             | string <DefaultValue>event_logo_default.png</DefaultValue> |
| <Name>catchPhrase</Name>            | string                                                     |

### Exemple de réponse

```json
{
  "id": 1,
  "name": "24h de l'INSA",
  "description": "le festival le plus charismatique de France",
  "startDate": "2019-05-17T18:00:00.000Z",
  "endDate": "2019-05-19T21:30:00.000Z",
  "pictureUrl": "event_logo_default.png",
  "catchPhrase": "Quand c'est gratuit c'est moins cher !"
}
```
